# 🎰 777 Slot Machine

משחק סלוט משין 777 מרהיב עם אנימציות חלקות ומסך מלא.

## ✨ תכונות

- **מסך מלא** - חוויית משחק אימרסיבית ללא הסחות דעת
- **3 גלילים** - עם אנימציות סיבוב חלקות ומרהיבות
- **2 מצבי משחק**:
  - **רגיל**: הגלילים נעצרים אוטומטית אחד אחד
  - **ידני**: שליטה מלאה - עצור כל גלגל בנפרד
- **אפקטים קוליים**: צלילים לכל פעולה (סיבוב, זכיה, אי-זכיה)
- **אפקט הבהוב** בזכיה
- **התאמה אישית**: העלה תמונות משלך במקום הסמלים הדיפולטים

## 🎮 שליטה

- **Enter** - התחל סיבוב (או עצור גלגל במצב ידני)
- **ד / S** - פתח מסך הגדרות
- **Escape** - סגור מסך הגדרות

## 🚀 הפעלה

1. פתח את הקובץ `index.html` בדפדפן
2. לחץ Enter להתחלת משחק
3. תהנה!

## 🎵 הוספת סאונד משלך

כרגע המשחק משתמש בצלילים סינתטיים. תוכל להוסיף קבצי MP3/WAV משלך:

1. צור תיקייה בשם `sounds`
2. הוסף קבצים: `spin.mp3`, `win.mp3`, `lose.mp3`
3. עדכן את הקוד ב-`script.js` לטעון את הקבצים
