# 🎉 מסך זכייה פשוט - Simple Win Screen

## סקירה כללית

תכונה חדשה המאפשרת להציג מסך זכייה פשוט ונקי עם טקסט באנגלית, במקום המסך המלא עם כל השלבים וה-QR code.

## תכונות עיקריות

### 1️⃣ מסך זכייה פשוט
- **כותרת ראשית**: "Congratulations!" (פונט ענק 5em)
- **כותרת משנה**: "Please collect your prize at the counter"
- **תמונת הפרס**: מוצגת במרכז עם מסגרת זהב
- **שם הפרס**: מוצג בגופן גדול (2.5em) בצבע זהב
- **כותרת תחתונה**: "THE FUTURE OF GAMING"
- **Powered by Playzone**: בתחתית המסך

### 2️⃣ עיצוב מקצועי
- רקע גרדיאנט כחול-סגול מרשים
- אנימציות והבהובים
- אפקט pulse על הכותרת הראשית
- צללים והארות לעומק
- רספונסיבי מלא למובייל

### 3️⃣ תמיכה ב-WhatsApp (אופציונלי)
- אם הוגדר מספר WhatsApp, מוצג גם במסך הפשוט
- QR code קטן יותר (200x200)
- טקסט בעברית: "סרוק לשליחת הודעת זכייה בוואטסאפ 📱"

### 4️⃣ הגדרות פשוטות
- צ'ק בוקס בהגדרות: "מסך זכייה פשוט (ללא WhatsApp)"
- הסבר ברור למשתמש
- שמירה אוטומטית ב-localStorage וב-Firebase

## איך זה עובד?

### לוגיקת תצוגה:
```
אם "מסך זכייה פשוט" מסומן:
  ├─ הצג מסך Congratulations פשוט
  └─ אם יש מספר WhatsApp:
      └─ הצג גם QR code במסך הפשוט

אם "מסך זכייה פשוט" לא מסומן:
  └─ הצג מסך מלא עם כל השלבים (רק אם יש WhatsApp)
```

## שימוש

### להפעלת מסך זכייה פשוט:
1. לחץ על מקש `S` או על כפתור ההגדרות
2. גלול לסעיף "מצב מסך זכייה"
3. סמן את "מסך זכייה פשוט (ללא WhatsApp)"
4. לחץ על "שמור הגדרות"
5. כשיש זכייה - יופיע מסך Congratulations פשוט!

### עם WhatsApp:
- אם יש מספר WhatsApp מוגדר, הוא יוצג גם במסך הפשוט
- QR code קטן יותר ונקי יותר
- מתאים למקרים שרוצים גם מסך פשוט וגם אפשרות WhatsApp

### ללא WhatsApp:
- מסך נקי לגמרי
- רק Congratulations, תמונת הפרס, ושם הפרס
- מושלם לאירועים פשוטים

## דוגמאות שימוש

### מקרה 1: אירוע פשוט
```
✅ סמן "מסך זכייה פשוט"
❌ אל תהזן מספר WhatsApp
→ תוצאה: מסך Congratulations נקי לגמרי
```

### מקרה 2: אירוע עם WhatsApp אבל מסך פשוט
```
✅ סמן "מסך זכייה פשוט"
✅ הזן מספר WhatsApp
→ תוצאה: מסך Congratulations + QR code קטן
```

### מקרה 3: אירוע מלא (ברירת מחדל)
```
❌ אל תסמן "מסך זכייה פשוט"
✅ הזן מספר WhatsApp
→ תוצאה: מסך מלא עם כל השלבים והוראות
```
